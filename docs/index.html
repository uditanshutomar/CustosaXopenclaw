Custosa sits as a transparent WebSocket proxy between your clients and the AI gateway. Every message is analyzed in real-time. Safe traffic flows through. Threats get blocked or held for human review.

FEATURES

🔍

MULTI-LAYER DETECTION

Three-layer analysis pipeline: regex pattern matching against 50+ injection signatures, heuristic structural analysis, and optional ML classifier.

📲

TELEGRAM APPROVAL

Human-in-the-loop for edge cases. Uncertain messages get sent to your Telegram bot with [ALLOW] and [DENY] buttons. 5-minute timeout, auto-deny.

⚠

TOOL RISK CLASSIFICATION

Automatically holds critical tool invocations like exec, shell, bash. High-risk tools (browser, web_fetch) also require approval.

🔠

UNICODE NORMALIZATION

Prevents homoglyph bypass attacks using Cyrillic/Greek lookalike characters, zero-width characters, and other Unicode tricks.

🧠

SESSION-AWARE CONTEXT

Detects multi-message attack patterns by maintaining session context. Catches distributed injection attempts across conversation turns.

🚫

ROLE ESCALATION GUARD

Blocks client-injected system/admin roles and detects instruction laundering through tool outputs. Based on OWASP LLM Top 10.

INSTALLATION

RECOMMENDED — SINGLE COMMAND INSTALL

$ curl -fsSL https://raw.githubusercontent.com/uditanshutomar/CustosaXopenclaw/main/install.sh | bash

Automatically installs Custosa, runs the setup wizard, configures your Telegram bot, and opens the dashboard.

INSTALL VIA HOMEBREW (macOS / Linux)

$ brew tap uditanshutomar/custosaxopenclaw $ brew install custosa $ custosa install

Uses the official Homebrew tap. The custosa install command launches the setup wizard.

BUILD FROM SOURCE

$ git clone https://github.com/uditanshutomar/CustosaXopenclaw.git $ cd CustosaXopenclaw $ pip install -e . $ custosa install

Clone the repo, install in editable mode, then run the setup wizard. Requires Python 3.10+.

CLI COMMANDS

$ custosa install Run setup wizard
$ custosa serve Start proxy server
$ custosa status Show protection status
$ custosa start Start background service
$ custosa stop Stop background service
$ custosa logs View security logs
$ custosa dashboard Open web dashboard
$ custosa test Test detection engine
